# Quick Start Guide - Fitness Corner Gym

Get the application running locally in under 2 minutes.

---

## 🚀 Commands Checklist

Run these commands in your project terminal:

```bash
# 1. Clone the project repository
git clone https://github.com/aqmmarshaikh/gym-app.git
cd gym-app

# 2. Install all dependencies
npm install

# 3. Copy default configurations
cp .env.example .env.local

# 4. Launch Next.js local development server
npm run dev
```

---
