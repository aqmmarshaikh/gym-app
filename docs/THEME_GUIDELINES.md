# Theme Guidelines - Fitness Corner Gym

Guidelines for theme customization and stylesheets.

---

## 🎨 Stylesheet Integrations

* **Tailwind Utility Mappings**: Standard custom properties are defined inside `src/app/globals.css`.
* **Dark Mode Enforcements**: The system is designed to use a black/gold aesthetic.
* **Component Theming Rules**: Always use Tailwind variables (e.g. `bg-black`, `border-white/10`, `text-gold`) to keep component designs consistent.
