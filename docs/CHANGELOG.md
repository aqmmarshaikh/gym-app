# Changelog - Fitness Corner Gym

All notable changes to the Fitness Corner Gym project are documented in this file.

---

## [1.1.0] - 2026-07-02

### Added
* **Gym Machine Management**: CRUD module inside Coach and Owner panels to manage available machinery.
* **Workout Assignment Engine**: Configured 17 default weekly programs with drag-and-drop / push-button exercise reordering.
* **Smart Routine Filtering**: Real-time filtering to exclude exercises requiring unavailable, disabled, or under-maintenance machinery from member views.
* **Digital Attendance Check-Ins**: Interactive front desk member scanning simulation.
* **Branding Tool**: Custom interface for owners to configure operational settings.
* **Enterprise CI/CD Pipelines**: Automatic building validation workflows configured via GitHub Actions.

---

## [1.0.0] - 2026-06-15

### Added
* Initial project release.
* Mobile-first responsive layout shell.
* Guest marketing landing page and WhatsApp booking dispatcher.
* Member portal containing digital ID cards, calorie logging widgets, and interactive rest timers.
* Offline demonstration support using LocalStorage state wrappers.
