# Technical Stack - Fitness Corner Gym

Detailed inventory of languages, libraries, databases, and frameworks utilized in this application.

---

## 🛠 Frameworks & Core Runtime

* **Next.js 15.0.3**: Utilizes App Router, server/client components, and static generation.
* **React 19.0.0**: State hooks, effect synchronization, and UI components.
* **TypeScript 5.x**: Strictly typed components, data models, and database interfaces.
* **Node.js**: Recommended runtime version is **20.x** (LTS).

---

## 🎨 UI & Styling Architecture

* **Tailwind CSS v4.0.0**: Used for responsive utility styling and layout designs.
* **Framer Motion 11.11.17**: Drives physical micro-interactions, layout transitions, and page load entry animations.
* **Lucide React 0.460.0**: Icons library used across member, coach, and owner modules.
* **Next Themes**: Controls CSS variable themes.

---

## 💾 Storage & Services

* **Firebase Web SDK 10.14.0**: Exposes client-side interfaces to Cloud Firestore, Authentication, and Cloud Storage.
* **LocalStore API**: Serializes structures directly into browser LocalStorage to facilitate 100% offline usage during demonstrations.

---

## ⚙️ DevOps & Tooling

* **Next Lint / ESLint 9.x**: Custom configurations to enforce code quality rules.
* **Prettier**: Automatic code formatting.
* **GitHub Actions**: Automated pipeline validating code builds.
