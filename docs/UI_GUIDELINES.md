# UI Guidelines - Fitness Corner Gym

Design system standards and aesthetics for FCG.

---

## 🎨 Theme Colors & Palette

* **Primary Background**: True black (`#000000`) and dark values (`#070707`, `#0A0A0A`) to ensure high contrast on OLED screen panels.
* **Secondary Surface**: Translucent dark surfaces mimicking a glass layout:
  * Class: `bg-white/[0.01]` or `bg-white/[0.02]`
* **Primary Accent Color**: Gold (`#D4AF37`) representing premium branding. Used on labels, buttons, and focused borders.
* **Semantic Tints**:
  * Success/Active: Light emerald green.
  * Warning/Maintenance: Amber orange.
  * Danger/Suspended: Bright red.
